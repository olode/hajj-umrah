Orange Palette Classes

Orange color palette contains 13 colors: 1 main orange color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any robust controls, that makes Robust very effective, flexible and configurable.

Please note: You can use .bg-orange and .btn-orange as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

Class Description
.bg-orange Orange Class for background color.
.bg-orange
.bg-lighten-*		 For lighten orange background color, this two classes needed. Here *: 1,2,3,4 lighten orange color options.
.bg-orange

.bg-orange
.bg-accent-*		 For accent orange background color, this two classes needed. Here *: 1,2,3,4 accent orange color options.
