Indigo Palette Classes

Indigo color palette contains 13 colors: 1 main indigo color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any robust controls, that makes Robust very effective, flexible and configurable.

Please note: You can use .bg-indigo and .btn-indigo as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

Class Description
.bg-indigo Indigo Class for background color.
.bg-indigo
.bg-lighten-*		 For lighten indigo background color, this two classes needed. Here *: 1,2,3,4 lighten indigo color options.
.bg-indigo
.bg-darken-*		 For darken indigo background color, this two classes needed. Here *: 1,2,3,4 darken indigo color options.
.bg-indigo
.bg-accent-*		 For accent indigo background color, this two classes needed. Here *: 1,2,3,4 accent indigo color options.
