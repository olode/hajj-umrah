Lime Palette Classes

Lime color palette contains 13 colors: 1 main lime color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any robust controls, that makes Robust very effective, flexible and configurable.

Please note: You can use .bg-lime and .btn-lime as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

Class Description
.bg-lime Lime Class for background color.
.bg-lime
.bg-lighten-*		 For lighten lime background color, this two classes needed. Here *: 1,2,3,4 lighten lime color options.
.bg-lime
.bg-darken-*		 For darken lime background color, this two classes needed. Here *: 1,2,3,4 darken lime color options.
.bg-lime
.bg-accent-*		 For accent lime background color, this two classes needed. Here *: 1,2,3,4 accent lime color options.
