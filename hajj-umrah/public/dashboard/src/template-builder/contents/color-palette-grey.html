Grey Palette Classes

Grey color palette contains 13 colors: 1 main grey color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any robust controls, that makes Robust very effective, flexible and configurable.

Please note: You can use .bg-grey and .btn-grey as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

Class Description
.bg-grey Grey Class for background color.
.bg-grey
.bg-lighten-*		 For lighten grey background color, this two classes needed. Here *: 1,2,3,4 lighten grey color options.
.bg-grey
.bg-darken-*		 For darken grey background color, this two classes needed. Here *: 1,2,3,4 darken grey color options.
.bg-grey
.bg-accent-*		 For accent grey background color, this two classes needed. Here *: 1,2,3,4 accent grey color options.
